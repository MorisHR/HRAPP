Accessibility Features

  • Keyboard Navigation: Use Arrow keys to move between tabs
  • Focus Management: Tab key moves focus, Enter/Space activates
  • Screen Readers: Full ARIA labels and roles
  • High Contrast: Visible outlines in high contrast mode
  • Reduced Motion: Respects prefers-reduced-motion setting
  • Focus Indicators: Clear 3px blue outline on focus
